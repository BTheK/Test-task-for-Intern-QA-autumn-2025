# Тестовое задание для QA стажёра Авито - Задание 2.1

В этом репозитории представлены тест-кейсы и автоматизированные тесты для тестируемого микросервиса.

## Структура проекта
* `TESTCASES.md` — Описание тест-кейсов.
* `test_api.py` — Код автотестов (Python + pytest).

## Инструкция по запуску

### 1. Требования
Необходимо, чтобы был установлен **Python 3** и добавлен в список переменных PATH.

### 2. Установка зависимостей
Откройте терминал, перейдите к папке с проектом (скриптом) и выполните команду для установки библиотек:

```bash
pip install requests pytest```

### 3. Запуск тестов
Для запуска тестового скрипта выполните в терминале команду:

```bash
pytest -v -s test_api.py```

Флаги:

-v: вывод результатов по каждому тест-кейсу.
-s: вывод принтов в консоль.
